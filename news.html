Науковиці Полтавської політехніки успішно вдосконалили навички володіння алгоритмами Data Science, пройшовши курси від ІТ-компанії Grid Dynamics та Харківського ІТ-кластеру

Науковиці Національного університету «Полтавська політехніка імені Юрія Кондратюка» Олена Двірна, Ксенія Вергал та Олена Гайтан успішно пройшли курси для викладачів технічних спеціальностей від ІТ-компанії Grid Dynamics та Харківського ІТ-кластеру за напрямом «Вступ до програмування мовою Python для Big Data та Data Science».

Упродовж трьох місяців інтенсивного навчання викладачі актуалізували знання методів та алгоритмів Data Science, а завершення курсу для учасників супроводжувалось здачею фінального проєкту щодо застосування механізмів інтелектуального аналізу для вирішення прикладних задач обробки великих даних та отриманням відповідних сертифікатів

Запрошуємо студентів навчально-наукового інституту інформаційних технологій та робототехніки пройти онлайн-курс «Базові правила безпеки в цифровому середовищі», розроблений фахівцями CRDF Global, експертною групою з кібербезпеки компанії EY в Україні та технічними експертами компанії Technomatix. Курс містить цікаві відео, практичні кейси та цікаві квізи, що буде цікаво як для початківців, так і для досвідчених користувачів.

Курс складається з 10 навчальних модулів і надає можливість:

  • ознайомитися з основними загрозами в сучасному інформаційному просторі;
  • з’ясувати найбільш поширені помилки, які роблять користувачів у цифровому просторі та способи їх уникнення;
  • опанувати базові правила захисту інформації на мобільних телефонах, комп’ютерах та в соціальних мережах;
  • навчитися розпізнавати фейкові новини;
  • дізнатися про основні способи захисту від кібершахрайства.

Курс є безкоштовним для студентів Полтавської політехніки. У результаті успішного проходження кожен з учасників отримає іменний сертифікат CRDF Global.

Для проходження курсу необхідно зареєструватись за посиланням: https://cybereducation.org/mc/index.php/usr/login/login, вказати університет та обрати спеціальність, на якій ви зараз навчаєтесь.
